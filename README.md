# Game-2048 - Репліка гри "2048" (з унікальними покращеннями)

### Мета
#### Розробка репліки популярної гри "2048" з деякими покращеннями:
 - Функція автоматичної гри з пошуком найкращих ходів
 - Функція зміни розміру ігрового поля: 4х4, 6х6, 8х8
 - Власний стиль компонентів
 - Аудіо супровід подій
 - Збереження найкращого результату в активній сесії браузера

### Стек технологій
#### Frontend проєкт (працює в браузері, не потребує серверу)
- Мова програмування: JavaScript
- Розмітка та стилі: HTML, CSS
